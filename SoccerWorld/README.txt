SoccerWorld website

1) To access the admin interface, go to /SoccerWorld/index.php
Here you can make changes to the database and such
Note: when adding a brand new item to the database, please remember to fill out all the boxes in the form and don't forget to attach an image :)

2) Main user website can be accessed at /SoccerWorld/home.html

3) So far, we can click on the Clothing tab and see a list of all the products in the db filtered by itemtype "clothing". Adding or editing the database will automatically update the products on the Clothing tab
